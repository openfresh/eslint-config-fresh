## Install

```bash
$ npm install --save-dev eslint-config-fresh
```

## Usage

In `.eslintrc`:

```json
{
  "extends": [
    "eslint-config-fresh"
  ]
}
```

In `package.json`:

```json
{
  "eslintConfig": {
    "extends": [
      "eslint-config-fresh"
    ]
  }
}
```
